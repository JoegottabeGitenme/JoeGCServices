Free, open weather data services powered by OGC standards. Access NOAA weather data through WMS, WMTS, and EDR APIs.

Services Online

🔌 Available Services

🌎 WMS - Web Map Service

OGC-compliant Web Map Service for rendering weather data as map images. Supports GetCapabilities, GetMap, and GetFeatureInfo operations.

View Capabilities →
🧮 WMTS - Tiled Map Service

High-performance tiled map service for efficient weather visualization. Pre-rendered tiles for fast loading in web mapping applications.

View Capabilities →
📈 EDR - Environmental Data

OGC API - Environmental Data Retrieval for point, area, and trajectory queries. Get weather data in GeoJSON or CoverageJSON formats.

Explore API →

📊 Available Data Sources

🌎 GFS - Global Forecast

Global Forecast System with 0.25 degree resolution. Temperature, wind, humidity, precipitation, and more with forecasts out to 384 hours.

🌦 HRRR - High Resolution

High-Resolution Rapid Refresh with 3km resolution and hourly updates. Best for short-term forecasts over CONUS.

🛰 MRMS Radar

Multi-Radar Multi-Sensor composite with 2-minute updates. Real-time precipitation and reflectivity over CONUS.

📡 GOES-18 Satellite

GOES West geostationary satellite imagery. Clean longwave infrared (10.3um) with 5-minute updates.

🚀 Coming Soon

🌞 GOES-16 East

GOES East satellite imagery covering the eastern US, Atlantic, and Gulf of Mexico.

🌊 NAM Model

North American Mesoscale model for regional forecasts with 12km resolution.

⛰ NWS Alerts

Real-time severe weather alerts, watches, and warnings from the National Weather Service.
