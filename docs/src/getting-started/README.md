## Deployment Options

Weather WMS supports multiple deployment methods:

1. **Docker Compose** (Recommended for development)
   - Quick local setup
   - All services in one command
   - Perfect for testing and development

2. **Kubernetes** (Recommended for production)
   - Horizontal scaling
   - High availability
   - Production-grade monitoring

3. **Helm Chart** (Production-ready)
   - One-command deployment to K8s
   - Configurable via values.yaml
   - Integrated monitoring stack

Choose the quick start guide to get up and running in minutes, or dive into the prerequisites and installation guide for a detailed walkthrough.
